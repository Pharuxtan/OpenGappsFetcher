# OpenGapps Fetcher

A script automatically fetch OpenGapps from SourceForge

## Structure
```
    |-- platform/architecture
        |-- version
            |-- variant: [Array] 
            |-- downloads
                |-- variant
                    |-- name
                    |-- date
                    |-- state
                    |-- size
                    |-- md5
                    |-- md5file
                        |-- name
                        |-- download
                    |-- sha1
                    |-- download
            |-- state
```
